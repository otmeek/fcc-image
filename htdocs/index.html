Search imgur galleries!

Usage

Searching for images. Simply include the search term(s) at the end of the URL

https://fcc-image-otmeek.c9users.io/imagesearch/cats

For pagination, include the following query parameter with the page number

https://fcc-image-otmeek.c9users.io/imagesearch/cats?offset=2

See the latest queries

https://fcc-image-otmeek.c9users.io/api/latest/imagesearch
